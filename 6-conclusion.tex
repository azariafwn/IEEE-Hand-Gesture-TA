\section{Conclusion}
This paper presented an optimized software architecture for real-time smart home control using a quantized LSTM network on a Raspberry Pi 5. By leveraging 3D hand landmarks and temporal sequence modeling, the system achieved a high classification accuracy of 98.5\% for individual gestures and a 99.1\% accuracy for sequential command execution. The implementation of 8-bit quantization proved critical, reducing the model's memory footprint to 1.07 MB and enabling a stable processing speed of 23.83 FPS.

Future work will focus on expanding the gesture vocabulary and implementing a more robust multi-hand tracking system to allow multiple users to interact simultaneously. Additionally, exploring light-weight Transformer-based architectures could potentially further enhance the temporal feature extraction for even more complex gesture sequences.
