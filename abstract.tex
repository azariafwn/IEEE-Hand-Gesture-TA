\begin{abstract}
While the Internet of Things (IoT) has accelerated smart home adoption, conventional interaction methods like mobile applications and voice commands remain limited by physical dependency and environmental noise. Hand gesture recognition offers an intuitive, contactless alternative; however, deploying complex Deep Learning models for real-time temporal sequence classification on resource-constrained edge devices poses significant computational challenges. This study proposes an optimized software architecture utilizing a Stacked Long Short-Term Memory (LSTM) network deployed on a Raspberry Pi 5. To ensure real-time viability, the model was optimized via 8-bit integer (INT8) post-training quantization. Experimental results demonstrate that this optimization strategy achieved a 97.7\% reduction in runtime RAM usage (from 47.22 MB to 1.07 MB) and an 89.7\% reduction in model size, with a negligible accuracy loss of only 0.25\%. The system achieved a high classification accuracy of 98.5\% for individual physical gestures and a 99.1\% macro-average accuracy for complex sequential command execution. Furthermore, the architecture maintains a stable processing speed of 23.83 FPS with an average edge inference latency of 28.82 ms. This ultra-low latency sits significantly below the 100 ms threshold recognized in Human-Computer Interaction (HCI) standards, guaranteeing a seamless and responsive user experience by ensuring the visual feedback loop remains imperceptible to the user. These results prove the system's robustness for "always-on" ambient intelligence in smart home environments.
\end{abstract}
